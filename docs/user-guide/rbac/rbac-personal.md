title: Personal

<img src="/img/rbac/rbac-personal.webp" alt="personal settings" width="1000" />

This screen can be accessed by all users and handles the following.

- **User Details** - Display the logged in user's email, display name and organization. Note that for non mult-tenant systems the organization is called **Ziggy Solo**.
- **Receive Alerts** - check to receive email alert notifications.
- **Change Password**
- **Assignments** - show Teams, Flows, Connections, Secrets and Messaging Items that the user has been assigned.
