---
title: Audit Logs
---

Ziggy maintains the following log/audit data. Upon request, you can access these via the CLI and REST API.

## Authentication Log
This logs successful and failed login attempts.

<img src="/img/rbac/rbac-auth-log.webp" alt="authentication log" width="1000" />

## Event Log
This lists 

- creation, updates and deletions for Flows, Connections, Secrets, Schedules and Messaging Items.
- User and team creation.
- Flow, connection, secret, schedule and messaging assignments to teams and users.

<img src="/img/rbac/rbac-event-log.webp" alt="event log" width="1000" />

## Access Denials
Shows any attempts to access backend services by unauthorized users.
