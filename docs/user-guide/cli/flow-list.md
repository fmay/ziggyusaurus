---
title: Flow List
---

`ziggy flow list  [--server FriendlyName] [--flow-ids] [--tags tag1 tag2] `

This lists the flow uuid, name and tags for each flow specified by `--flow-ids` and/or `--tags`.

For example, the command `flow list -tags 'tagtest' 'scheduled'`
